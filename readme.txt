----------------Puslapio Vartotojai:-------------------------
------------------- Pirkėjas:
-- Homepage'e galės:
   - matyti naujausias pridėtas prekes (max 3)
   - matyti banerius pagal produktų kategorijas (juos paspaudus nukreips į visų produktų sąrašą, kur jau bus išfiltruotos prekės pagal pasirinktą banerio kategoriją).
   - užpildyti subscribe to newsletter formą.

-- Įėjęs į All Products puslapį galės: 
   - matyti visus produktus ir filtrą
   - išsifiltruoti produktus pagal kategorijas
   - paspausti ant produkto ir patekti į sigle item puslapį.

-- Įėjęs į single item puslapį galės:
  - matyti visą informaciją apie jį
  - įsidėti prekę į krepšelį/į whishlist

-- Paspaudęs ant krepšelio icon:
 - matys pridėtos prekės pavadinimą, kiekį, jos kainą
 - galės ištrinti prekę, pakeisti prekės kiekį, tęsti užsakymą, užpildyti user information ir pateikti užsakymą

------------- Puslapio admin'as:
-- Prekių stock management
-- /admin puslapyje:
   - naudodamas formą galės pridėti produktą
   - galės matyti naujienlaiškiui užsiregistravusius vartotojus ir jų email adresus.
   - galės matyti pateiktus užsakymus lentelėje/ order management




Important:
- Mobile First – designing for all the mobile devices.
- Layout is responsive.
- Site is compatible with popular browsers.
- HTML or CSS codes are validated.
- Favicon is uploaded.
- App icons display well, especially on iOS mobile devices.
- Website Security 
- Live Chat Integration
- Well-designed home, category, product, checkout pages.
- Company logo is linked to the site’s homepage.
- Submitted data is being sent to the user’s email or phone number.
- Forms are sending data correctly.
- Thank you page displays after hitting the “submit” button.
